## 概要

NVIDIA OptiX Toolkit(OTK)は、OptiX・CUDA runtime・CUDA driver APIにまたがる統一的なエラーチェックマクロと、デバイス側のピンポイントデバッグ出力機構を提供し、GPUレイトレーシングアプリケーションの診断を容易にする。DemandPbrtSceneのサンプルで実際の利用例が示されている。
