## 概要

Azure Storageが提供するBlob、Files、NetApp Files、Queue、Table、Diskなど各データサービスを整理し、ネットワーク侵入制御からコンピュート層、ストレージ層、プラットフォームサービスに至る基本的な参照アーキテクチャを示す。用途別の技術選定ガイドや、ハイブリッドファイル共有・エンタープライズDR・メインフレームデータ移行など本番運用可能なアーキテクチャ集への入り口としてまとめられている。
