## 概要

Azure Virtual Desktopをエンタープライズ規模で展開するためのランディングゾーンアーキテクチャを解説する。ID管理・ネットワーク・セキュリティ・運用監視など9つの設計領域を軸に、ガバナンスを保ちながらスケールする基盤の作り方を示す。
