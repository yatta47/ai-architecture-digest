## 概要

Microsoft Learnが提供する、Azure上でクラウドネイティブなアプリケーションを設計する際の指針をまとめたリファレンス記事。自己修復・冗長化・水平スケーリング・パーティショニングといったコア原則から、運用性やビジネス要件に基づく設計まで、信頼性・拡張性・保守性を高めるための考え方を体系的に整理している。特定の実装事例ではなく、設計時に参照すべき原則集としての位置づけ。
