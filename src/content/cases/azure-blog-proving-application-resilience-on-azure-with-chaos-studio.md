## 概要

Microsoft Azureは、実運用で観測された障害パターンを模したシナリオ単位でカオステストを実行できる新サービス「Azure Chaos Studio Workspaces」をパブリックプレビューで公開した。ゾーン停止やDNS障害、ID基盤障害などの名前付きシナリオを対象リソースグループに対して自動推奨・実行でき、プラットフォーム層とアプリケーション層の両方の耐障害性を本番障害の前に検証できる。AI/LLM技術ではなく、インフラの回復力検証を目的としたカオスエンジニアリング機能の告知である。
