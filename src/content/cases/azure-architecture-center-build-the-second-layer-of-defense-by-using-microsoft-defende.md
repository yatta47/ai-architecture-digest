## 概要

ランサムウェアなど高度な脅威に対する多層防御シリーズの第3弾。予防的統制をすり抜けた攻撃を早期に検知するため、Defender for Endpoint/Identity/Office 365/Cloud Appsのシグナルを Defender XDR で相関させ、統合インシデントとして自動調査・封じ込めを行うアーキテクチャを解説する。
