## 概要

Azure Architecture Centerが公開する、Google Cloud（旧GCP）とAzureのサービスをカテゴリ別に対応付けたリファレンス記事。データベース、データウェアハウス、ETL、ビッグデータ分析、時系列/IoT、AI/機械学習など多岐にわたる領域で、両クラウドのほぼ同等なサービスを一覧化している。マルチクラウド構成の計画やGoogle CloudからAzureへの移行検討に使うことを想定している。
