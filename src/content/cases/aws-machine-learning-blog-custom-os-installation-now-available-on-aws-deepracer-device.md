## 概要

AWS DeepRacerはサポートが終了したUbuntu 16.04/20.04でしか起動できず、実験や研究の継続が難しくなっていた。新たに公開された開発者向けブートローダー(shimベース)により、独自の証明書管理でカスタムOSや第三者ディストリビューションを安全に起動できるようになった。開発者モードはMorse信号やHDMI警告で明示され、標準構成への復元も可能なため、既存ハードウェアの寿命を伸ばせる。
