## 概要

Google Cloudに精通したエンジニア向けに、Azureのアカウント・サブスクリプション・リソースグループの管理階層と、Google Cloudの組織・プロジェクト構造との対応関係を解説するガイドである。管理グループ、サブスクリプション、リソースグループという4階層の構成や、Azure RBAC、Azure Resource Manager、CLI/PowerShell/ポータルなど管理インターフェースの違いも紹介している。両プラットフォームは類似の機能セットを持つが、リソースの組織方法や用語には明確な違いがあると説明する。
