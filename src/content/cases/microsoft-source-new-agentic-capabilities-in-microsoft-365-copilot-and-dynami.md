## 概要

Microsoft 365 CopilotとDynamics 365にSales AgentとService Agentが一般提供開始され、MCP(Model Context Protocol)のツール/アプリを共通基盤にエージェント型AIが日常の業務フローに組み込まれた。Work IQとDynamics 365データを基に、営業担当者は顧客・商談コンテキストを即座に把握しCRM更新を自動化し、サービス担当者はケース要約と次善アクション提示で対応を迅速化する。IDCの調査では生成AI投資1ドルあたり平均3.70ドルのリターンが得られるとされ、Sandvik CoromantやNorthern Trustなどの顧客事例も紹介されている。

## 設計のポイント

- MCP(Model Context Protocol)のツール/アプリを共通基盤とし、Microsoft 365とDynamics 365の間でエージェントの振る舞いと文脈理解を一貫させる
- Work IQで実際の業務パターンを学習し、CRMデータと組み合わせて顧客・商談コンテキストを自動的に収集する
- 商談後の要点・反論・フォローアップ事項を自動でCRMフィールドに反映し、データ品質と追跡精度を高める
- Outlook/Teamsなど既存の業務アプリの中にエージェントを組み込み、システム切り替えの摩擦を減らす

## 使いどころ

- 商談準備や過去のやり取りの把握に時間を取られている営業チーム
- 案件情報がシステム間に散在しCRM入力が後回しになりがちな営業組織
- ケース対応のスピードと一貫した顧客対応が求められるカスタマーサービス部門
- 既存のMicrosoft 365/Dynamics 365環境を活かしたままエージェントAIを段階的に導入したい企業
