## 概要

オンプレミスのAzure Local上にAzure Virtual Desktopのセッションホストを展開し、Microsoft Entra IDとオンプレADDSのハイブリッドID同期、FSLogixによるプロファイル管理、Azure Arcによる統合ガバナンスを組み合わせるアーキテクチャを解説する。
