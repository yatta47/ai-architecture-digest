## 概要

Azure上でWordPressを運用する際のセキュリティとパフォーマンスに関するベストプラクティス集。Azure Front DoorとWeb Application Firewallによる防御、不要なプラグイン/テーマの削除、静的コンテンツのCDNオフロードとキャッシュ無効化、二要素認証の導入、XML-RPCや管理画面へのアクセス制限などを解説する。ホスティング方式(VM/App Service等)によらず適用できる共通の強化策を提示する。
