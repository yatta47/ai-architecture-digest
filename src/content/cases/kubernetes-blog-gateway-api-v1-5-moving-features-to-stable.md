## 概要

Gateway API v1.5では、ListenerSetやTLSRoute、HTTPRoute CORS Filterなど6つの実験的機能がStandard（GA）チャンネルへ昇格した。今後はリリーストレイン方式を採用し、機能凍結日までに準備が整った機能のみを出荷することで、より安定したリリースサイクルを目指す。
