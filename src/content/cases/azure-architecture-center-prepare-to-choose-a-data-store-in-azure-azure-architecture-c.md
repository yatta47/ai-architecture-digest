## 概要

クラウド移行のランディングゾーン設計にあたり、データ形式・アクセスパターン・コスト・セキュリティなど多角的な要件からAzureのデータストアを選定するための準備プロセスを解説する。リレーショナルDB、Dataverse、キー・バリューストアなど選択肢ごとの判断基準を示す。
