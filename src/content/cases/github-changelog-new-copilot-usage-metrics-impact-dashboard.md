## 概要

GitHubは企業管理者・組織オーナー向けに、Copilotの利用状況を可視化する新しい「インパクトダッシュボード」を提供開始した。利用者を「コード中心」「エージェント中心」「マルチエージェント/Copilotアプリ」「非活性(ライセンス取得済みだが未活用)」というAI導入フェーズごとのコホートに分類し、コホートごとのPRマージ数・マージ速度・利用者比率・1日あたりコード行数などの指標や、非活性層と活性層の生産性比較、半年間のトレンド推移、導入を深化させるための推奨アクションを提示する。管理者は単純な利用者数の把握を超えて、開発者がCopilotのどの機能をどれだけ活用しているかという定着度のストーリーを把握できる。
