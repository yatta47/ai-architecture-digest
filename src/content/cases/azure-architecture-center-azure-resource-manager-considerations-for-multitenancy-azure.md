## 概要

マルチテナントソリューションでAzure Resource Managerを使う際の要点を解説するガイド。BicepやARMテンプレート、デプロイスタック、テンプレートスペックによるIaCでテナントごとのリソースをプロビジョニングし、Azure RBACやタグでテナント分離とコスト配分を管理、リソースクォータ制限への対処法(サブスクリプション分割やクォータ申請)も示す。
