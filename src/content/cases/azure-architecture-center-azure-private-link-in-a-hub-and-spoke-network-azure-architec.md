## 概要

ハブアンドスポーク型ネットワークトポロジでAzure Private Linkを使い、PaaSリソースにプライベートエンドポイント経由でアクセスする方法を解説する。ネットワークアーキテクトやクラウドソリューションアーキテクトを対象読者とし、コスト削減と一元的なセキュリティ管理を両立するハブアンドスポーク構成の利点を整理する。
