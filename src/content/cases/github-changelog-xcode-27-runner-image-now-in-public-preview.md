## 概要

GitHub HostedのmacOSランナーでXcode 27がパブリックプレビューとして利用可能になった。イメージのサポートモデルをOSバージョン単位からXcodeメジャーバージョン単位に変更し、新しいツールチェーンとの互換性をCI上で早期に検証できるようにする。
