## 概要

キャッシュ・レスポンスモッキング・開発者ポータルなどを備えたAPIゲートウェイであるAzure API Managementを、自社のマルチテナントアプリケーションのAPI提供にどう活用するかを整理した記事。単一インスタンスで複数テナントを扱う分離モデルの選択肢を示す。
