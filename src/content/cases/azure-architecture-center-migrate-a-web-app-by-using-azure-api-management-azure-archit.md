## 概要

旅行業界のEコマース企業が、Azure API Managementを新旧APIのファサードとして配置し、レガシーなオンプレミスWebアプリのフロントエンドを段階的にAzure PaaSへ移行するシナリオ。フロントエンドを変更せずに裏側のサービスを順次モダナイズできるため、既存機能を維持しながら再開発の手間を最小化できる設計を解説している。
