## 概要

オンプレミス/マルチクラウド上のSQL Serverインスタンスを、Azure Arc対応サーバーおよびAzure Arc対応SQL Serverによって一元的に管理・監視・ガバナンスするリファレンスアーキテクチャを示す。AKS on Azure LocalでコンテナSQL Managed Instanceをホストするケースも含める。
