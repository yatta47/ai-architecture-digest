## 概要

Azure Architecture CenterによるDevOps入門ガイド。継続的インテグレーション/デリバリー/デプロイ/監視という基本概念と、Azure DevOps・Azure Pipelines・GitHub Actions・Azure Monitorなどのサービスを整理し、GitHubリポジトリからPR承認・マージを経てAKSへデプロイする典型的なワークフロー例を示す。あわせてCI/CDパイプラインやGitOps、DevSecOps、ブルーグリーンデプロイなど発展的なアーキテクチャ記事群への入り口となっている。
