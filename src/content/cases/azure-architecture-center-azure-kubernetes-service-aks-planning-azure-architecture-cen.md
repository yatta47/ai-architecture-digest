## 概要

Azure Kubernetes Service (AKS) を導入する組織向けに、設計と運用のロードマップを解説するガイド記事。多くの標準的なワークロードには事前設定済みのAKS Automaticを推奨し、ネットワークやノードプールを細かく制御したい場合はAKSベースラインクラスタを参照実装とするAKS Standardを選ぶという判断軸を示す。マイクロサービス、高セキュリティ規制、災害復旧など特化シナリオ別のベースライン実装にも言及する。
