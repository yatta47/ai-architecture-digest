## 概要

Ingress-NGINXの2026年3月退役を控え、Ingress設定をGateway APIへ変換するツールIngress2Gatewayが1.0に到達。30種類以上のNGINXアノテーションを翻訳し、実クラスタでの挙動一致を検証する統合テストと、変換不能な設定への警告を備える。
