## 概要

ゲノミクスや気象シミュレーションなど大規模計算ワークロード向けに、Azure BatchやCycleCloudを使ったHPC基盤の構築方法を整理したガイド。オンプレミスとのハイブリッドバースティングやクラウドネイティブ構成の比較も含む。
