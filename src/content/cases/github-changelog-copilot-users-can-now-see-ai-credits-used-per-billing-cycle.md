## 概要

GitHub Copilot BusinessおよびEnterpriseユーザー向けに、GitHub設定のCopilot使用状況ページで、個別予算が未設定でも当該billingサイクル中に使用したAIクレジット量を確認できるようになった。従来は予算に対する消費率（%）のみの表示だったため、個別予算を持たない多くのユーザーは実際の月間使用量を把握できなかった。管理者が予算を設定している場合は、予算総量に対する消費クレジット数が表示される。
