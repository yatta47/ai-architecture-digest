## 概要

AWSとAzureにおける「リソース」の概念、リソースグループの違い(Azureは削除でリソースも削除される等)、デプロイ手段(ポータル/CLI/ARM/Bicep/Terraform)、タグ付け仕様の差を比較し、AWSからAzureへの移行者向けに要点を整理する。
