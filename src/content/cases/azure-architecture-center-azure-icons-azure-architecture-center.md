## 概要

Azure Architecture Centerが提供する、アーキテクチャ図やドキュメント作成に使う公式Azureアイコン集のダウンロードページ。SVG形式で配布され、Visioなど任意の作図ツールに取り込める一方、アイコンの改変や自社製品への流用は禁止されている。2026年7月更新ではAI Gateway関連やMicrosoft Foundry関連のアイコンなど10種以上が追加された。
