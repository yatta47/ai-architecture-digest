## 概要

マイクロサービスの境界を組織図やレイヤーではなくビジネスドメインから導く戦略的DDDの進め方を、ドローン配送サービスを例に解説する。境界づけられたコンテキストごとにユビキタス言語を持たせ、疎結合かつ凝集度の高いサービス設計を目指す。
