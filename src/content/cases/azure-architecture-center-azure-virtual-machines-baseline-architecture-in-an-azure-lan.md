## 概要

プラットフォームチームが集中管理するAzureランディングゾーンに、VMベースのワークロードを統合する際の責任分担と設計変更を解説。Firewall・Bastionなどプラットフォーム所有リソースと、サブネット・Managed Disksなどワークロードチーム所有リソースの境界を明確化する。
