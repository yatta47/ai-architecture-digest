## 概要

Azure Architecture Centerによる、クラウド時代のアイデンティティ/アクセス管理(IAM)設計への入門ガイド。オンプレミスのネットワーク境界に頼る従来型のセキュリティモデルに代えて、Microsoft Entra IDを中核に据えた認証・認可の考え方を示し、オンプレミスADとのハイブリッド統合やマルチテナントID、クロスクラウドIDなど個別の実装アーキテクチャ集への入り口として機能する。
