## 概要

Kubernetes向けUIツールHeadlampに、Cluster API（CAPI）リソースを可視化する新プラグインが追加された。Cluster・MachineDeployment・MachineSet・Machine・MachinePoolなどの状態やオーナーシップ階層をダッシュボードやマップビューで確認でき、UI上からのスケール操作やブートストラップ設定の閲覧も可能になる。CNCFのLFXメンターシッププログラムの一環としてHeadlampコミュニティと共同開発され、現在はAlphaリリースとして提供されている。
