## 概要

Azure Architecture CenterのGet Startedガイドは、オンプレミス・クラウド・エッジにまたがるシステム統合を、API Management・Logic Apps・Service Bus・Event Grid・Functions・Data Factoryなどのサービス群でどう設計するかを整理する。同期的なAPI呼び出しと非同期のメッセージング/イベントの使い分け、およびワークフローのオーケストレーションを軸に、基本的な統合アーキテクチャ図と関連するリファレンスアーキテクチャ・技術選定ガイドへの導線を提供する。
