## 概要

GitHub ProjectsのフィルタバーがAND/ORを使った高度な検索式に対応し、質問ごとに別々のビューを維持しなくても1つのビューで柔軟に絞り込めるようになった。あわせてPRのレビュー状態によるフィルタ（reviews:）や、90日を超えたデプロイステータスの自動削除ポリシーも導入された。
