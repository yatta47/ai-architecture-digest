## 概要

Logic AppsとAPI Managementのみを使い、SaaSやAzureサービス、既存Webサービスといったバックエンドシステムへの呼び出しをオーケストレーションする基本的なエンタープライズ統合の参照アーキテクチャ。Microsoft Entra IDによる認証を起点に、APIゲートウェイとワークフローが連携する構成を示す。
