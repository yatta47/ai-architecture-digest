## 概要

Azure App ServiceやApplication Gateway、WAF、Azure Monitorなどを組み合わせて、可用性・拡張性・セキュリティを備えたWebアプリを構築するための入門ガイド。ゾーン冗長なApp Service構成やプライベートエンドポイントを使ったベースラインアーキテクチャを示しつつ、災害復旧・API保護・WordPressホスティングなど用途別の発展アーキテクチャ一覧へ橋渡しする。
