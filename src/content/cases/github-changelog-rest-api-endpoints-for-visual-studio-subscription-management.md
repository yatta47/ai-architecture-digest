## 概要

GitHub Enterprise Cloudの管理者が、Visual StudioサブスクリプションとGitHubアカウントの紐付けをREST API経由で取得・登録・解除できるようになった。VSSのUPN形式がSCIM IDと一致せず自動マッチングできない組織向けに、UPNとGitHubハンドルの対応表を使った一括登録スクリプトが可能になる。
