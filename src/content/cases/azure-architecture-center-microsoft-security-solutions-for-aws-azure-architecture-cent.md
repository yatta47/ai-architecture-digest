## 概要

本ガイドは、Microsoft Entra、Defender for Cloud Apps、Defender for Cloud、Purview、Sentinel、Defender XDR、Security CopilotといったMicrosoft製品群を用いて、ゼロトラスト原則に基づく多層防御(defense-in-depth)をAWS環境に適用する方法を解説する。アイデンティティ管理、脅威検知・対応、データガバナンス、コンプライアンス報告までをAWSと横断的に統合する構成を示している。
