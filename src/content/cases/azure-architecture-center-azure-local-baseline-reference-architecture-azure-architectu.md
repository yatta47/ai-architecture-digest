## 概要

Azure Local 2311以降を用いてオンプレミス/エッジで可用性の高い仮想化・コンテナ基盤を構築するためのワークロード非依存な参照アーキテクチャ。物理ノードのクラスタ設計、ストレージ、ネットワークの選定指針と、Azure Arc・Key Vault・Monitor・Defender for Cloudなどによるハイブリッド管理の統合方法を示す。
