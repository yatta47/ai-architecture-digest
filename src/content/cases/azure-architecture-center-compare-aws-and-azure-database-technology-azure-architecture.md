## 概要

AWSのAmazon RDSやRedshift、DynamoDBなどのデータベース・分析サービスと、Azure SQL DatabaseやMicrosoft Fabric、Cosmos DB、Azure Databricksなどの対応するAzureサービスをマッピングしたリファレンスガイド。課金モデルの違いや、リレーショナルDB・NoSQL・データウェアハウス・キャッシュ・移行ツールの各カテゴリでの代替サービスを整理し、AWSからAzureへのデータベース移行を計画するアーキテクトを支援する。
