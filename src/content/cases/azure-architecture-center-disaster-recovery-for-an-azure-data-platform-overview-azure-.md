## 概要

Azureエンタープライズデータプラットフォームのディザスタリカバリ(DR)戦略を設計するためのシリーズ第1弾。物理的な災害からのサービス復旧に焦点を絞り、ランサムウェア対策やフェイルバック、上流/下流システムの復旧はスコープ外と明示した上で、コスト・信頼性・複雑性のトレードオフを検討する前提を整理する。
