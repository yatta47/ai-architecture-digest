## 概要

Azure Architecture Centerによるマイクロサービスのコンテナオーケストレーションガイド。多数のコンテナ化されたマイクロサービスインスタンスを本番規模で運用する際のスケーリング・負荷分散・望ましい状態の維持といった課題を整理し、AKS・Container Apps・Container Instances・Azure Red Hat OpenShiftなどAzure上のオーケストレーション選択肢を比較している。
