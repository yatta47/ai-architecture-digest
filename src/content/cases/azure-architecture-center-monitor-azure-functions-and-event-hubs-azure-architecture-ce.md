## 概要

Application InsightsをAzure FunctionsとEvent Hubsの組み合わせに統合し、Application Mapやエンドツーエンドのトランザクション詳細で処理時間や障害箇所を可視化する監視手法を解説する。イベント処理システムの健全性把握と障害の早期発見を目的とする。
