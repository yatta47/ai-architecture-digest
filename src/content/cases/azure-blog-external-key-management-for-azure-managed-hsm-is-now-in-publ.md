## 概要

Azure Key Vault Managed HSMに、鍵materialを顧客が所有・運用するハードウェア(オンプレミスや信頼できるサードパーティ)に置ける外部鍵管理機能がパブリックプレビューで追加された。規制やデータ主権要件でクラウド外に鍵を保持する必要がある政府・金融・重要インフラ向けの選択肢。
