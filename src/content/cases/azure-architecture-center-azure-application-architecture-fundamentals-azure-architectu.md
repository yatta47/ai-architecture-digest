## 概要

Azure Architecture Centerによる、クラウドアプリケーション設計の基本方針をまとめたガイド記事。オンプレミス設計(モノリシック・同期処理・MTBF重視)とクラウドネイティブ設計(分散・非同期・耐障害性重視・IaC)の違いを整理し、Well-Architected Frameworkの5つの柱(信頼性・セキュリティ・コスト・運用・パフォーマンス)に沿って設計判断を行うことを推奨している。組織のクラウド導入標準(Landing Zoneなど)への準拠や、アーキテクチャスタイル・データストアモデルの選定手順にも触れている。
