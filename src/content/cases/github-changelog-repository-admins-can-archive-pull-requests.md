## 概要

リポジトリ管理者がPull Requestを削除せずにクローズ・ロックした状態でアーカイブし、一般公開から隠せる機能が追加された。スパムや不適切なPRのモデレーション、削除が望ましくない法務・ポリシー要件への対応、監査目的での履歴保持に使える。
