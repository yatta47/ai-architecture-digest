## 概要

Azure Filesは、AIモデルの重みをGPUレプリカ間で共有ロードするAI推論、Kubernetes向けRWX共有ストレージ、オンプレミスNFSワークロードの移行という3つのモダンLinuxワークロードのシナリオに向けた新機能をまとめて提供する。nconnectやゾーン配置、Provisioned v2課金、1サブスクリプションあたり最大1万ファイル共有などにより、起動の高速化とコスト最適化を図る。研究プラットフォームZooniverseやヘルスケア関連製品大手Medlineの移行事例が紹介されている。
