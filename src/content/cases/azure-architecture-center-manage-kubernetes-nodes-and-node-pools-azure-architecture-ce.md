## 概要

AKSとAmazon EKSにおけるノード・ノードプールの管理方法を比較する。EKSマネージドノードグループやKarpenterによる動的なノードスケーリング、Kata Containersによるセキュアなコンテナ実行環境などの選択肢を解説する。
