## 概要

Siemens Teamcenter PLM（製品ライフサイクル管理）をAzure上で全社統合運用するベースラインアーキテクチャ。クライアント・Web・エンタープライズ・リソースの4層構成を単一の仮想ネットワーク内に配置し、複数拠点に分散していたPLMインスタンスを集約してデータサイロを解消する。
