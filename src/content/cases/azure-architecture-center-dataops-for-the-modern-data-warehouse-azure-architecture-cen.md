## 概要

架空の都市計画局(Contoso)を題材に、モダンデータウェアハウス(MDW)パターンに基づくエンドツーエンドのデータパイプラインとDevOps/DataOpsプロセスを解説するリファレンスシナリオ。Azure Data FactoryとData Lake Storageでデータを取り込み・検証し、Databricksで整形した後、Synapse Analyticsへロードしてデータサイエンティストのモデル学習とPower BIによる可視化の両方に活用する構成を示している。
