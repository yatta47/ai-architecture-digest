## 概要

Azure Load TestingでJMeterのカスタムプラグインを使い、Event HubsやIoT Hubへのユーザー/デバイス挙動を模擬する負荷テストのソリューション例を示す。FunctionsとEvent Hubsを通じたデータフローに対しKPIを設計し、ダッシュボードで結果を監視・分析する構成を解説する。
