## 概要

Azure Architecture Centerが、マイクロサービス化に伴うデータ整合性・サービス間通信・障害の連鎖・レガシー移行という課題に対応する代表的な設計パターンをまとめた記事。Anti-Corruption Layer、Bulkhead、Saga、Sidecar、Strangler Figなど、それぞれ異なる関心事に対応するパターン群と、APIゲートウェイ関連のGatewayパターンを紹介している。
