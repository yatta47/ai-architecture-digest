## 概要

SAP HANA/NetWeaverのAzure移行・高可用化を、SUSEのSAP自動化ソリューション(SLES4SAP)を使って手作業のデプロイ・構成ステップを削減しながら実現する構成を示す。モジュール化・再利用可能なビルディングブロックにより、複雑なHA構成でも構築ミスのリスクを抑える。
