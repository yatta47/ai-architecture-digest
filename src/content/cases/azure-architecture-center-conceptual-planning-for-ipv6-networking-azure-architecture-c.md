## 概要

IPv4アドレス枯渇に備え、Azure仮想ネットワークをIPv4からIPv6へ移行するための戦略を解説する。デュアルスタック構成により既存のIPv4サブネットにIPv6アドレスブロックを追加し、段階的に移行できる点を説明している。
