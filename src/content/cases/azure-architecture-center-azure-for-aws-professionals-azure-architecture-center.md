## 概要

AWSに精通したエンジニアがAzureの基本的なアカウント構造やサービスを理解するための対応表・入門記事群を紹介する。AWSのアカウントとAzureのサブスクリプションの違いや、コンピューティング・ストレージ・データベース・DevOps・監視などカテゴリ別にAWSサービスとAzureサービスの対応関係を整理し、マルチクラウド設計やAWSからAzureへの移行検討の出発点として使えるようにしている。
